# 移动端 Github 多端应用

基于 Taro3 + React 实现。

## 技术栈

- Taro
- React Hooks
- TypeScript
- SCSS

## 支持特性

- 🐑 查看自己的资料
- 搜索
  - [x] 用户
  - [ ] 项目
- 用户相关
  - [x] 资料
  - [x] 项目
  - [x] 关注
  - [x] 粉丝
- 登陆
  - [ ] github 登录
- 💰 更多特性持续迭代中...
